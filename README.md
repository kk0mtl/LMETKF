# Local Modulated Ensemble Transform Kalman Filter (LMETKF)

This repository provides the code used for the Lorenz-96 data assimilation experiments described in the following manuscript:

**An Improved Local Ensemble Transform Kalman Filter with Model Space Covariance Localization**

The materials include the Lorenz-96 model code, implementations of the LETKF, GETKF, LMETKF, LMETKF-Z, and LMETKF-noLoc algorithms, processed experimental outputs, and Python scripts for analyzing the results and generating the figures. The experiments used synthetic observations generated from the Lorenz-96 nature run; no external observational datasets were used.

## Figure generation

### Figures 1 and 2

The correlation-matrix diagnostics used in Figs. 1 and 2 can be generated by setting `debug_plot=True` in `enkf.py`.

### Figures 3, 6, and 7

Panels (a)–(e), which show the domain-averaged background state estimates of the tested filters together with the domain-averaged true state, were generated using the `plot_state_mean_timeseries()` function in `L96enkf.py`.

Panel (f), which shows the domain-averaged background RMSEs of the tested filters, was generated using `fig_rmse_timestep.py`. Before running this plotting script, the background RMSE `.npz` files must be generated using the `save_rmse_b()` function in `L96enkf.py` and stored in the `./rmse_npz` directory.

### Figures 4 and 5

The rank histograms and covariance diagnostics can be generated by setting `plot_save=True` in the main section of `L96enkf.py`.

The corresponding output files are:

- `8_2_rank_histogram_all_state_iEnKF_*` for the rank histograms
- `4_covariance_enkf_method_*` for the covariance diagnostics

Figures 4 and 5 were assembled from these saved outputs.

### Figures 8–10

The plotting procedures and time-mean data assimilation results used to generate Figs. 8–10 are contained in `case3_ens.py`. For each filter and ensemble size, the DA time, background RMSE, and background ensemble spread are organized into separate arrays before plotting.
